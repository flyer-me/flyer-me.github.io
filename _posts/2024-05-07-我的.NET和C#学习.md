# C#

C# 是一种 高效、开源、跨平台，面向对象的编程语言。

[C# 语言文档](https://learn.microsoft.com/zh-cn/dotnet/csharp/)

# .NET Core

.NET Core 是跨平台的、由强大的 C# 为编程语言的、高性能的应用程序平台，主要用于Web、云、游戏开发、物联网、本机应用。
建议从 [.NET 文档](https://learn.microsoft.com/zh-cn/dotnet/) 开始了解。

## ASP.NET Core

### [ASP.NET Core MVC - Dot Net Tutorials](https://dotnettutorials.net/course/asp-net-core-tutorials/)

ASP.NET Core MVC 包括 ASP.NET Core Web 技术特点，MVC，数据传递方式，模型绑定，数据注释，过滤器，HTML/Tag Helpers，状态管理，
以及 ORM 技术 EF Core。本文的部分翻译版本在[这里](https://github.com/flyer-me/PersonalLearnWay/tree/master/DotNet/aspnetcore)

### [EF Core](https://github.com/flyer-me/PersonalLearnWay/tree/master/DotNet/EF%20Core)

EF Core 在我的 PersonalLearnWay 项目中整理了详细的笔记。

### [ASP.NET Core Identity - Dot Net Tutorials](https://dotnettutorials.net/course/asp-net-core-identity-tutorials/)

ASP.NET Core Identity 包括用户和角色管理，认证与授权，密码策略，权限控制，第三方登录，内置安全等。
